# [V△NECKZER◉]:TRANSMISSION_ARCHIVE

A static GitHub Pages-ready redesign for `vaneckzero.com`: an atmospheric media archive for glitch art, live visual systems, CRT/projection work, and cybersecurity research.

## Structure

- `index.html` contains the homepage sections and editable archive entries.
- `styles.css` contains the visual system, responsive layout, and ambient CRT/signal treatments.
- `scripts/main.js` contains the subtle noise canvas, sticky header state, and archive filtering.
- `assets/media/transmission-hero.png` is a generated placeholder hero image. Replace it with your own still or video poster when ready.
- `research/` contains static research article stubs that can be expanded independently.

## GitHub Pages

1. Push this repo to GitHub.
2. In the repository settings, enable Pages.
3. Set the source to the `main` branch and root folder.
4. Point `vaneckzero.com` to the GitHub Pages site using your domain DNS settings.

For video, add a compressed `.mp4` or `.webm` to `assets/media/` and replace the hero `<picture>` in `index.html` with a muted looping `<video>` element.
